# MZ-2000用CMU-800インタフェース基板

　MZ-2000拡張ユニットに挿入できるCMU-800インタフェース基板を作ってみました。

![1](https://github.com/yanataka60/MZ-2000_CMU800/blob/main/JPEG/1.JPG)

![2](https://github.com/yanataka60/MZ-2000_CMU800/blob/main/JPEG/2.JPG)

![3](https://github.com/yanataka60/MZ-2000_CMU800/blob/main/JPEG/3.JPG)

　PC-8001_SDとの連結には秋月の分割ロングピンソケットFHU-2x42SGを使ってますが、若干高さが足りなくてCMU-800のコネクタを挿し込むと基板が少し斜めに浮き上がります。
