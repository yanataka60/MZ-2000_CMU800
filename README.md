# PC-8001用CMU-800アダプタ基板

　CMU-800はPC-8001本体後部の拡張端子にそのまま直結できるようですが、他になにも繋げられなくなってしまいます。

　拙作PC-8001_SDと両立を図るためアダプタ基板を作ってみました。

　PC-8001_SDと両立を図るために作ったものですが、フラットケーブル用のピンヘッダをつけてフラットケーブル->カードエッジの変換基板として使うなどの使い方もあると思います。

![3](https://github.com/yanataka60/PC-8001_CMU800/blob/main/JPEG/3.JPG)

　PC-8001_SDとの連結には秋月の分割ロングピンソケットFHU-2x42SGを使ってますが、若干高さが足りなくてCMU-800のコネクタを挿し込むと基板が少し斜めに浮き上がります。
