# MZ-2000用CMU-800インタフェース基板

　MZ-2000拡張ユニットに挿入できるCMU-800インタフェース基板を作ってみました。

　MZ-2000及びMZ-2200用として作成していますが、MZ-80B、MZ-2500もカードスロットの規格は同じなので使えると思います。

![1](https://github.com/yanataka60/MZ-2000_CMU800/blob/main/JPEG/1.JPG)

　スロットカバーを取り付けるとスロットカバーの穴がスロットより若干上にずれているため基板が上反り気味になりますが、動作に支障はないようです。
